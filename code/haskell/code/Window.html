Description
A bare-metal window manager, part of OpenGLINT. The manager draws windows into GraphicsCmd's as implemented in the OpenGLINT graphics-interface layer: at the moment that layer only translates to OpenGL commands.

Message functions
sendMessage :: Window -> Message -> Window

Send the message to the window and its children, in effect invoking the messageHandler function of each window with the body of the message supplied in the parameter.

This function is the main entry point into the window system. Messages should be created by external processes (such as a GLUT-driven program) and sent to the top-level window. The messages will be forwarded to child windows as appropriate, and any child-responses that need attention will be further processed, before finally returning a new top-level window

sendMessageTo :: Window -> ID -> MessageBody -> Response
Send the message to the specified window. Not really recommended, you should use sendMessage with a correctly addressed message instead, but it's useful if you need a Response instead of an updated Window.

pickWindows :: Window -> GraphicsPosition -> [(ID, Coordinate)]
Given a window and a point, calculates ID of windows containing the point, and the relative position within each window of the point. Returns the results in reverse order, i.e. the deepest-level window is at the front of the list.

Messaging types
type MessageHandler = Window -> MessageBody -> QueryFunction -> Response
A message handler takes a window and message and returns a response to the message. The message handler is usually a partially-applied function that has a parameter representing state of the window.
type QueryFunction = ID -> StateHandle -> StateValue
A QueryFunction is passed to windows during a message-send to allow them to query the state of other windows
type StateHandle = Int
A StateHandle is a unique identifier used by each window to refer to all or part of the state of the window; i.e. its interpretation is window-specific. For example, a window can use the value '0' to mean its complete state, or just a part of that state
