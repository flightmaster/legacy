SetPlanCheckpoints [Checkpoint]Set the current plan according to the list of supplied checkpoints
SetPlanIdents [Ident]Set the current plan according to the list of supplied identifiers. The system must be able to un-ambiguously resolve the first identifier from the list otherwise the operation fails. Subsequent ambiguities in the identifiers list are resolved by choosing the object closet to the preceding identifier.
